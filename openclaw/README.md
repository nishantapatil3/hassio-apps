# Home Assistant Add-on: OpenClaw

[![Release](https://img.shields.io/github/v/release/nishantapatil3/hassio-apps?filter=openclaw*)](https://github.com/nishantapatil3/hassio-apps/releases)

AI agent gateway with web UI by OpenClaw.

## About

OpenClaw provides an AI agent gateway with a built-in web interface for
interacting with LLM-powered agents. It supports direct provider API keys
and can integrate with LiteLLM for unified model routing.

## Features

- Web-based chat UI with agent interaction (port 18789)
- Multi-provider support (OpenAI, Anthropic, Gemini, OpenRouter, Google)
- LiteLLM proxy integration for unified model access
- Persistent sessions and workspace
- Configurable gateway authentication
